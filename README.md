## ✨ Tentang Viper AI

**Viper AI** adalah aplikasi web AI modern yang dibangun dengan arsitektur cepat dan ringan. Dirancang untuk memberikan pengalaman interaksi dengan kecerdasan buatan yang responsif, Viper AI memanfaatkan **Google Gemini API** sebagai otak di balik layar, dengan sentuhan kreatif dari **Dev** (aldan.my.id) dalam pengembangan antarmuka dan pengalaman pengguna.

Proyek ini sepenuhnya **open source** — kamu bebas menggandakan, memodifikasi, dan mendistribusikan ulang sesuai dengan lisensi yang berlaku.

### 🎯 Fitur Utama

- ⚡ **Responsif & Cepat** — Dibangun dengan Vite untuk performa terbaik
- 🧠 **AI-Powered** — Menggunakan Google Gemini API untuk pemrosesan bahasa alami
- 🌐 **Modern Stack** — TypeScript, ES Modules, dan arsitektur modular
- 🚀 **Siap Deploy** — Konfigurasi Vercel sudah tersedia (`vercel.json`)
- 🎨 **UI/UX Elegan** — Dikurasi oleh Gemini AI dengan sentuhan desain modern
- 🔒 **Environment Variables** — Keamanan API key terjamin

## 📦 Prasyarat

Sebelum memulai instalasi, pastikan kamu memiliki:

- **Node.js** (versi 18.x atau lebih baru) — [Download](https://nodejs.org)
- **npm** atau **yarn** atau **pnpm** — Package manager
- **Git** — Untuk cloning repository
- **Akun Vercel** (gratis) — Untuk deploy
- **API Key Google Gemini** — [Dapatkan di sini](https://ai.google.dev/)
